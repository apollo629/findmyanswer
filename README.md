# findmyanswer - Quite simple implementation of Spring Data JPA with RESTfull webapp

## How to Install and Run
mvn clean install
mvn jetty-run war
Create your local db with etc/initdb.sql file
sh etc/initial-data.sh
http://localhost:8080/questions

## Used Technologies
* Spring Data JPA
* Spring WEB
* Spring MVC
* Hibernate
* Postgre SQL
* H2 DB (test)
